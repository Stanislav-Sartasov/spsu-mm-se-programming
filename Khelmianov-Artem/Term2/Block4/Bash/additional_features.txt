Дополнительные фичи:
"..." - как в баше позволяют использовать переменные внутри
'...' - внутри только текст
< [file] - перенаправление ввода в файл
> [file] - перенаправление вывода в файл
$(...) - command substitution

Валидная команда:
cat < $(pwd)/file | wc | xargs $(a=ec b=ho echo $a$b) "wc output:" | cat > another_file
                         ^^^^^внешняя команда
